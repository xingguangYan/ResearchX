# ResearchX — Literature Search Quick Reference

## Web Search Query Templates

Use these patterns for effective literature discovery via `web_search`:

### Pattern 1: Review papers (best for overview)
```
web_search "[topic] review survey 2024 2025"
web_search "[topic] comprehensive review deep learning"
```

### Pattern 2: SOTA methods
```
web_search "[topic] state-of-the-art 2025 2026"
web_search "[topic] benchmark comparison"
web_search "[topic] outperforms existing"
```

### Pattern 3: Method-specific
```
web_search "[topic] [method family] 2024"
web_search "[method name] [application domain]"
```

### Pattern 4: Gaps and challenges
```
web_search "[topic] challenge limitation problem future direction"
web_search "[topic] open problem unsolved"
```

### Pattern 5: Emerging trends
```
web_search "[topic] emerging trend 2025 2026"
web_search "[topic] foundation model large language model"
web_search "[topic] new paradigm 2025 2026"
```

### Pattern 6: Cross-domain
```
web_search "[method from domain A] applied to [user's domain]"
web_search "[adjacent field] [method type]"
```

### Pattern 7: Datasets
```
web_search "[topic] dataset benchmark 2024 2025"
web_search "[topic] public dataset"
```

## Search Strategy Tips

1. **Iterative refinement**: Start broad, then narrow based on results
2. **Multiple angles**: Search from method, data, and application perspectives
3. **Temporal filter**: Focus on 2023-2026 for the core search, include 2020-2022 for foundational work
4. **Language**: Search in English + user's native language for comprehensive coverage
5. **Citation chaining**: When you find a good paper, search for papers that cite it
